# 《Rust 程序设计语言》

_作者：Steve Klabnik、Carol Nichols 和 Chris Krycho，以及 Rust 社区的贡献者_

本书假定你使用 Rust 1.97.0（发布于 2026-07-09）或更高版本，并在所有项目的 *Cargo.toml* 文件中使用 `edition = "2024"`，以启用 Rust 2024 Edition 的惯用写法。请参阅[第 1 章的“安装”部分][install]<!-- ignore -->，了解安装或更新 Rust 的方法；并参阅[附录 E][appendix-e]<!-- ignore -->，了解 Edition。

HTML 版本可在线阅读，网址为[https://doc.rust-lang.org/stable/book/](https://doc.rust-lang.org/stable/book/)，也可以通过 `rustup` 安装的 Rust 离线阅读；运行 `rustup doc
--book` 即可打开。

本书还提供了若干社区[译本][translations]。

本书还提供 [No Starch Press 出版的平装书和电子书][nsprust]。
