# 深入了解 Cargo 和 Crates.io

到目前为止，我们只使用了 Cargo 最基本的功能来构建、运行和测试代码，但它
还能做更多事情。本章将讨论它的一些其他高级功能，并展示如何完成以下任务：

- 通过发布配置文件自定义构建。
- 将库发布到 [crates.io](https://crates.io/)<!-- ignore -->。
- 使用工作空间组织大型项目。
- 从 [crates.io](https://crates.io/)<!-- ignore --> 安装二进制文件。
- 使用自定义命令扩展 Cargo。

Cargo 能做的事情远不止本章介绍的这些。要完整了解它的所有功能，请参阅[其
文档](https://doc.rust-lang.org/cargo/)。
